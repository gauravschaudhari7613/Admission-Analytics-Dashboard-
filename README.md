# Admission Analytics Dashboard 📊

Interactive Power BI dashboard created to analyze and visualize college admission data using dynamic charts, KPIs, and filters. The dashboard provides insights into branch-wise admissions, category distribution, gender ratio, cutoff comparison, fees analysis, and yearly admission trends.

---

## 🚀 Features
- Total Admissions KPI
- Category-Wise Admissions Analysis
- Branch-Wise Admission Comparison
- Admission Type Analysis
- Gender Distribution
- Branch Fees Comparison
- Cutoff Analysis by Branch
- Year-Wise Admission Trends
- Interactive Filters & Slicers

---

## 🛠 Tools & Technologies
- Power BI
- Microsoft Excel
- DAX
- Data Visualization

---

## 📈 Dashboard Insights
- CSE branch recorded the highest admissions.
- CAP admissions dominated management admissions.
- Gender distribution remained nearly balanced.
- Cutoff comparison highlighted high-demand branches.
- Admission trends varied across different academic years.
